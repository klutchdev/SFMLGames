### 1. Brick Hero

#### Instructions

##### Navigate to the main directory
`cd SFML_TUTORIAL`

##### Compile
`g++ -g ./BrickHero/main.cpp ./BrickHero/Game.cpp -o ./BrickHero/prog -lsfml-graphics -lsfml-window -lsfml-system -lsfml-audio -lsfml-network`

##### Run
`./BrickHero/prog`
